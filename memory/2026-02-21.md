## 游戏移动端触摸延迟修复

批量修复了多个 React 游戏的移动端触摸延迟问题（300ms click delay）。

### 修复的游戏

1. **DinoGame.tsx** (恐龙快跑)
   - 添加 `onTouchStart` 处理，消除 300ms 延迟
   - 优化游戏循环依赖，使用 ref 避免频繁重建定时器

2. **FlappyGame.tsx** (像素鸟)
   - 同上修复方案

3. **FroggerGame.tsx** (青蛙过河)
   - 添加 3 处 `onTouchStart` 处理

4. **SimonGame.tsx** (Simon 记忆游戏)
   - 添加 `onTouchStart` 处理

5. **WhackAMoleGame.tsx** (打地鼠)
   - 添加 `onTouchStart` 处理

6. **ZigzagGame.tsx** (之字形)
   - 添加 `onTouchStart` 处理

### 技术方案

```tsx
// 修复前
<canvas onClick={jump} />

// 修复后
<canvas 
  onClick={jump}
  onTouchStart={(e) => {
    e.preventDefault();
    jump();
  }}
/>
```

### 构建状态
- 所有修复已通过 TypeScript 编译和 Vite 构建验证
